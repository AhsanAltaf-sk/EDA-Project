This repository contains my **first completed task** for the **Data Analytics Internship at CodeAlpha**.

## 📊 Analysis Performed

The following steps were completed in this EDA project:

### 1. Dataset Overview
- Displayed sample rows using `df.head()`
- Checked dataset dimensions using `df.shape`
- Reviewed column names using `df.columns`

### 2. Data Understanding
- Examined data types using `df.dtypes`
- Used `df.info()` to inspect structure and completeness
- Used `df.describe()` for statistical summary

### 3. Data Quality Checks
- Checked for missing values
- Observed that the dataset contains **no missing values**

### 4. Categorical Analysis
- Analyzed the target variable distribution
- Examined gender distribution
- Compared heart disease occurrence across categories

### 5. Visual Analysis
- Correlation heatmap
- Boxplots for outlier detection
- Histogram for age distribution
- Countplots for:
  - heart disease by gender
  - heart disease by chest pain type

## 📈 Key Insights

Some important findings from this project include:

- The dataset is **clean and complete**, with no missing values
- Most patients fall in the **middle-age to older-age group**
- Heart disease appears to vary based on:
  - **age**
  - **gender**
  - **chest pain type**
  - **other clinical features**
- Some numerical columns contain **potential outliers**
- Certain variables show stronger relationships with the target than others
